# MetroScan

MetroScan is an Android application developed to explore and analyze metro-related information through a simple and intuitive mobile interface. The project focuses on building a structured and scalable Android application using modern development practices and clean architecture.

It serves as a foundation for future enhancements related to location-based services, transit analysis, and smart mobility systems. The application emphasizes modular design, maintainable code, and practical implementation of Android development concepts.

---

## Tech Stack

- Language: Kotlin  
- Platform: Android  
- IDE: Android Studio  
- UI: XML Layouts & Jetpack compose
- Architecture: MVVM / Modular Design  
- Build System: Gradle  
- Libraries: AndroidX
